## L5核心要素清单

### 五维评估标准
1. **📦 数据完整性**: 完整数据叙事、对比基准线、单位来源
2. **🎨 视觉规范**: 品牌级配色、发光/特效精准、渐变立体感
3. **📐 信息层级**: 标题+副标题完整叙事、一眼命中核心结论
4. **🧠 认知摩擦**: 零摩擦、盲测100%解码、行动指引清晰
5. **✨ 设计精度**: 品牌出版级、每个像素有意图、圆角阴影精细

### 通用L5技术要素
- `markLine`: 目标参考线、均值线
- `markPoint`: Callout气泡标注、TOP标记
- `markArea`: 事件区间高亮、目标区间
- `shadowBlur`: 发光效果(10-20)
- `tooltip增强`: YoY/环比对比、完整数据故事
- `副标题`: 核心洞察前置、行动建议

## L5升级模式总结

### 通用代码模板

#### 1. 目标参考线
```javascript
markLine: {
  silent: true,
  data: [{
    yAxis: target,
    lineStyle: {color: t.positive, width: 2, type: 'dashed'},
    label: {formatter: `目标 ${target}`, position: 'end', ...}
  }]
}
```

#### 2. Callout气泡标注
```javascript
markPoint: {
  data: [{
    coord: [x, y],
    value: 'TOP\n' + value,
    symbol: 'circle',
    symbolSize: 50,
    itemStyle: {
      color: color + '22',
      borderColor: color,
      borderWidth: 2
    },
    label: {color: color, fontSize: 10, fontWeight: 600}
  }]
}
```

#### 3. 发光效果
```javascript
itemStyle: {
  shadowBlur: 15,
  shadowColor: color + '66'
}
```

#### 4. YoY对比tooltip
```javascript
tooltip: {
  formatter: params => {
    const yoy = ((curr - prev) / prev * 100).toFixed(1);
    return `当前: ${curr}<br/>去年: ${prev}<br/>YoY: <span style="color:${yoy>0?'#30D986':'#FF6B6B'}">${yoy>0?'+':''}${yoy}%</span>`;
  }
}
```
