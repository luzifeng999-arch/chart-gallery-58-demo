# 📊 信息可视化图表库 Chart Gallery

58种信息可视化图表完整图库 - 10大类别，涵盖比较、趋势、构成、分布、关系、流动、地理、层级、网络、文本信息类图表。

## ✨ 特性

- 🎨 **58种图表类型** - 覆盖10大语义类别
- 🌓 **双主题支持** - Obsidian Dark + Aurora Light 一键切换
- 📱 **响应式设计** - 支持桌面端和移动端
- 🔍 **搜索功能** - 快速定位图表类型
- 🎯 **统一图库** - 单页面切换浏览所有图表
- 📈 **实时编辑** - 部分图表支持数据编辑面板

## 📂 图表分类

### 01. 比较类 Comparison (8种)
柱状图、条形图、分组柱状图、雷达图、热力矩阵、子弹图、哑铃图、斜率图

### 02. 趋势类 Trend (7种)
折线图、面积图、堆叠面积图、阶梯图、蜡烛图、甘特图、流图

### 03. 构成类 Composition (7种)
饼图、环图、堆叠柱状图、旭日图、矩形树图、瀑布图、百分比堆叠

### 04. 分布类 Distribution (7种)
直方图、散点图、箱线图、气泡图、小提琴图、热力日历、平行坐标

### 05. 关系类 Relationship (4种)
相关矩阵、弦图、散点矩阵、邻接矩阵

### 06. 流动类 Flow (6种)
桑基图、漏斗图、流程图、力导向图、旅程地图、时间轴

### 07. 地理类 Geographic (5种)
填色地图、气泡地图、地理热力图、流向地图、点密度图

### 08. 层级类 Hierarchy (5种)
树图、思维导图、组织架构图、冰柱图、径向树

### 09. 网络类 Network (4种)
弧图、网络邻接矩阵、力导向网络、知识图谱

### 10. 文本信息类 Text/Info (5种)
KPI卡片、仪表盘等

## 🛠️ 技术栈

- **图表库**: ECharts 5
- **样式**: 原生 CSS + CSS Variables
- **主题**: Dark/Light Mode
- **架构**: 纯静态 HTML

## 📖 使用方法

